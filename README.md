Simple React app that is a Clash Royale themed parody of the Rice Purity Test
Deployed on:
https://clash-royale-purity-test.vercel.app/

How to deploy this app:
1. Clone the repository
git clone https://github.com/your-username/clash-royale-purity-test.git

2. cd clash-royale-purity-test

3. Install dependencies
npm install

4. Run locally (development)
npm run dev

The app will run at http://localhost:5173/
 (check your terminal for exact URL)

npm run build

Generates an optimized production build in the dist/ folder
